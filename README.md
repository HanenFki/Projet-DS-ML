# Medical Cost Prediction Project

Machine learning project using regression models to predict medical insurance charges.

## Models
- Linear Regression
- Ridge
- Lasso
- ElasticNet

## Feature Selection
- Backward Elimination
- Lasso
- RFE

## Evaluation Metrics
RMSE, MAE, R², Adjusted R², MAPE, Cross-validation

## Result
Feature selection improves simplicity without major performance loss.
